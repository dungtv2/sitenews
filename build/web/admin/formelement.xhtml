SIZE OPTIONS
Kích cỡ của form input

Bạn chỉ cần thêm class (.label-lg|.label-sm) cho label, class (.input-lg|.input-sm)cho input.

Code Ví dụ :
<div class="form-group form-group-default required">
    <label class="label-lg">Lớn</label>
    <input type="text" class="form-control input-lg" placeholder="lớn" required="required" />
</div>

Radio Option
Color Options

Bạn chỉ cần thêm class radio. Nếu muốn radio có màu sắc thì bạn sẽ sử dụng các class sau đây
.radio-primary, .radio-complete, .radio-success,.radio-warning, .radio-danger, .radio-info

Code ví dụ :
<div class="radio radio-success">
   <input type="radio" name="optionyn" value="1" id="yes" />
   <label for="yes"> Yes</label>
   <input type="radio" name="optionyn" checked="checked" value="2" id="no" />
   <label for="no">No</label>
</div>

State Radio

Bạn chỉ cần thêm thuộc tính disabled="disabled" cho radio cần

Date and Dropdown Controls
Date Picker

   <input type="text" class="form-control date" />
   <span class="input-group-addon date">
     <i class="i-bl i-m2" />
   </span>
</div>
